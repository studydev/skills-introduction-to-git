# Git 소개

_커맨드 라인(CLI)과 VS Code를 사용하여 게임 프로젝트에 Git 버전 관리를 적용해보세요._

## 환영합니다

- **대상**: Git 버전 관리를 배우고 싶은 초보 개발자
- **배울 내용**: 커밋, 브랜치, 히스토리, 협업 기초를 포함한 Git의 핵심 개념
- **만들 것**: Git 저장소를 만들고, 샘플 코드를 추가한 뒤, 간단한 기능을 개발합니다.
- **사전 요구사항**:

  - Git이나 버전 관리에 대한 사전 경험이 필요하지 않습니다.
  - 권장: 커맨드 라인 인터페이스(CLI)에 대한 기본적인 이해
  - 권장: Visual Studio Code에 대한 기본적인 이해

- **소요 시간**: 이 실습은 60분 이내에 완료할 수 있습니다.

이 실습에서 다음을 수행합니다:

1. 버전 관리가 무엇인지, 개발자가 왜 사용하는지 이해합니다.
1. Git 사용자 정보를 설정합니다.
1. 첫 번째 저장소를 만들고 커밋합니다.
1. 프로젝트 히스토리를 확인하고 파일 변경 사항을 비교합니다.
1. 브랜치를 사용하여 안전하게 실험합니다.
1. Git 협업 개념을 배웁니다.
